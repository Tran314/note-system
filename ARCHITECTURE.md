## 技术栈

### 前端 (React)
- **框架**: React 18 + TypeScript
- **构建**: Vite 5
- **样式**: TailwindCSS
- **状态**: Zustand
- **编辑器**: TipTap
- **表单**: React Hook Form + Zod

### 存储
- **对象存储**: Tencent COS
- **用途**: 笔记附件、图片、文件上传

### 桌面端 (Electron)
- **框架**: Electron 28
- **打包**: electron-builder
- **更新**: electron-updater
- **存储**: electron-store

---

## 数据流

```
用户操作 → React 前端 → Tencent COS SDK → COS 存储
```

---

## 安全设计

1. **JWT 双 Token**: Access Token (短时效) + Refresh Token (长时效)
2. **密码加密**: bcrypt 哈希存储
3. **COS 签名上传**: 临时密钥授权上传
4. **输入验证**: Zod Schema 验证
5. **CORS 配置**: 严格的跨域策略

---

## 性能优化

1. **代码分割**: 按需加载，减少首屏时间
2. **COS CDN**: 全球加速分发
3. **虚拟滚动**: 大量数据流畅渲染
4. **防抖节流**: 优化高频操作
