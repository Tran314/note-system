## 技术栈

### 后端 (NestJS)
- **框架**: NestJS 10 + TypeScript
- **数据库**: PostgreSQL 16 + Prisma ORM
- **缓存**: Redis 7
- **认证**: JWT + Passport.js
- **文档**: Swagger/OpenAPI

### 前端 (React)
- **框架**: React 18 + TypeScript
- **构建**: Vite 5
- **样式**: TailwindCSS
- **状态**: Zustand
- **编辑器**: TipTap
- **表单**: React Hook Form + Zod

### 桌面端 (Electron)
- **框架**: Electron 28
- **打包**: electron-builder
- **更新**: electron-updater
- **存储**: electron-store

---

## 部署架构

### Web 部署
```
Nginx → Frontend (React) → Backend API (NestJS) → PostgreSQL/Redis
```

### 桌面端部署
```
Electron App → Embedded Backend → SQLite/Local Storage
```

---

## 安全设计

1. **JWT 双 Token**: Access Token (短时效) + Refresh Token (长时效)
2. **密码加密**: bcrypt 哈希存储
3. **请求限流**: 防止暴力破解
4. **CORS 配置**: 严格的跨域策略
5. **输入验证**: Zod  Schema 验证
6. **SQL 注入防护**: Prisma ORM 参数化查询

---

## 性能优化

1. **代码分割**: 按需加载，减少首屏时间
2. **数据库索引**: 优化查询性能
3. **Redis 缓存**: 热点数据缓存
4. **虚拟滚动**: 大量数据流畅渲染
5. **防抖节流**: 优化高频操作
